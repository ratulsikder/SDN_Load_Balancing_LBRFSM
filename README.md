A switch-migration framework named Load Balancing via Relaxed and Faster Switch Migration (LBRFSM) for the ONOS (Open-Network Operating System) SDN Controller.

This is the implementation of the algorithm shown in the Research Article: "A relaxed and faster switch migration framework to balance the load of distributed control plane in software defined networks"

Cite the article:

Sikder, R., Hossain, M.S., Alenazi, M.J.F. et al. A relaxed and faster switch migration framework to balance the load of distributed control plane in software defined networks. Cluster Comput 27, 8995–9012 (2024). https://doi.org/10.1007/s10586-024-04445-z
